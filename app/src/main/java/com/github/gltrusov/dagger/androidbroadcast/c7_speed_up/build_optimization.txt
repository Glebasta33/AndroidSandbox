Ускорение (и оптимизация) сборки на Dagger.

1. Форматирование сгенерированного кода.
kapt {
    arguments {
        args("dagger.formatGeneratedSource", "enabled")
    }
}
Форматирование скорее замедляет сборку, но лучше помогает понять сгенерированный код.

2. Оптимизация сгенерированного кода для ускорения инициализации (запуска) приложения.
kapt {
    arguments {
        args("dagger.fastInit", "enabled")
    }
}
